# REST Countries (restcountries)

Free REST API providing comprehensive data about world countries including name, capital, currencies, languages, flags, population, and geographic information. Covers 250+ countries and dependencies with 80+ fields per country, updated from 35+ authoritative sources.

**APIs.json:** [https://raw.githubusercontent.com/api-evangelist/restcountries/refs/heads/main/apis.yml](https://raw.githubusercontent.com/api-evangelist/restcountries/refs/heads/main/apis.yml)

### REST Countries API

RESTful API providing normalized country data including names, capitals, currencies, languages, borders, flags, population, memberships, and geographic information for 250+ countries.

- **Human URL:** [https://restcountries.com](https://restcountries.com)
